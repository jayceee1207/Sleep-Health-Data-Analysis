### Sleep-Health-Data-Analysis
An exploratory data analysis (EDA) of 100,000 sleep records using Python, Pandas, and Seaborn.

## Sleep Health & Lifestyle Analysis 🛌📊
# Project Overview
This project is an Exploratory Data Analysis (EDA) of a dataset containing 100,000 records of sleep health and lifestyle metrics. The goal of this analysis is to uncover relationships between sleep duration, daily habits (like caffeine intake), and cognitive performance across different professions.

# Tools & Libraries Used
Python

Pandas (Data cleaning and manipulation)

Matplotlib & Seaborn (Data visualization)

Jupyter Notebook

# Key Insights Discovered
1. The Caffeine Cliff: Visualized a clear drop in median sleep duration as caffeine intake approaches 400mg using boxplots.

2. The "Brain Power" Correlation: Uncovered a 0.9 positive correlation between Sleep Quality Score and Cognitive Performance Score using a correlation heatmap.

3. Busting the "Doctor" Myth: Filtered data to compare Students and Doctors, revealing that the Doctors in this dataset suffer from significantly lower sleep quality and cognitive performance than Students.

4. Age and Risk: Demonstrated that the median age jumps significantly (to near 60 years old) for individuals classified with a "Severe" sleep disorder risk.
